# CodeAlpha Frontend Development - Task 1: Image Gallery

## Project Name
**CodeAlpha_ImageGallery**

## Features
- Responsive image gallery using HTML and CSS grid
- JavaScript lightbox preview
- Next / Previous image navigation
- Hover effects and smooth transitions
- Category filters (Nature, City, Animals, Travel)
- Mobile-friendly layout

## Files
- `index.html` → structure of image gallery
- `style.css` → design, responsiveness, hover effects
- `script.js` → lightbox and filter logic

## How to Run
1. Keep all files in the same folder.
2. Open `index.html` in browser.

## Note
Images are loaded from online URLs for quick preview.
If you want, you can later replace them with your own images.
